# W3SEO Onpage SEO Framework

A practical Onpage SEO framework by **Đoàn Trình Dục** for building content that is clear, useful, search-friendly, and ready for modern search experiences such as Google Search, AI Overview, and answer engines.

This repository documents the W3SEO workflow for keyword research, LSI keyword mapping, outline building, content writing, keyword placement, EEAT signals, internal linking, and WordPress onpage optimization.

## Core Purpose

The goal of this framework is to turn SEO content creation into a repeatable process:

1. Find the main keyword and LSI keywords
2. Build a detailed SEO outline
3. Write content based on the outline
4. Insert SEO keywords naturally
5. Improve EEAT with experience, expertise, authoritativeness, and trustworthiness

## W3SEO Content Flow

### 1. Keyword Layer

Start with one clear primary keyword, then build supporting LSI keywords, semantic entities, and related questions.

### 2. Outline Layer

Create a detailed outline before writing. The outline should match search intent, user questions, and entity coverage.

### 3. Content Layer

Write the content based on the outline. Each section should solve a specific user need.

### 4. SEO Placement Layer

Insert the primary keyword and LSI keywords naturally into the title, introduction, headings, body, image alt text, FAQ, and conclusion.

### 5. EEAT Layer

Add practical experience, expert explanation, author/reviewer context, source transparency, and verification guidance.
