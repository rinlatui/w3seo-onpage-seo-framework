# W3SEO Onpage SEO Framework

A practical Onpage SEO, GEO, EEAT, Semantic SEO, and WordPress publishing framework by **Đoàn Trình Dục**.

This repository documents a repeatable SEO workflow for building content that is useful for users, understandable for search engines, and easier for AI answer engines to extract.

## Core Purpose

The goal of this framework is to turn Onpage SEO into a repeatable process instead of doing SEO by feeling.

```text
Keyword → Intent → Outline → Content → EEAT → Semantic SEO → Information Gain → GEO → Internal Link → Schema → Publish → Refresh
```

## Quick Start

1. Start with `docs/01-w3seo-process.md`
2. Map keywords with `docs/02-keyword-mapping.md`
3. Analyze intent with `docs/03-search-intent-analysis.md`
4. Build an outline from `templates/article-outline-template.md`
5. Review the content with `checklists/onpage-seo-checklist.md`
6. Score the page with `checklists/content-quality-scoring-sheet.md`
7. Publish using `checklists/wordpress-publishing-checklist.md`
8. Refresh older pages using `checklists/content-refresh-checklist.md`
