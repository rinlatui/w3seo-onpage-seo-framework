# Onpage SEO Checklist

## Keyword

- [ ] Main keyword selected
- [ ] LSI keywords mapped
- [ ] Related entities identified
- [ ] Search intent confirmed
- [ ] Keyword mapping completed

## Content Structure

- [ ] One H1 only
- [ ] H2 sections follow user intent
- [ ] H3 sections support H2 topics
- [ ] Introduction answers the main problem
- [ ] Conclusion includes trust and next step

## Semantic SEO

- [ ] Entity coverage checked
- [ ] Related questions added
- [ ] Information Gain added
- [ ] Content hub connection identified

## EEAT

- [ ] Experience signal included
- [ ] Expert explanation included
- [ ] Author profile connected
- [ ] Reviewer box added when needed
- [ ] Sources or verification guidance included
- [ ] Last updated date visible

## Internal Links

- [ ] Links to related articles
- [ ] Links to service/money page where relevant
- [ ] No redirected internal links
- [ ] No empty anchor text
- [ ] Hub/cluster relationship checked

## Schema

- [ ] Article schema considered
- [ ] FAQ schema considered
- [ ] Person/Author schema connected
- [ ] Organization schema connected
- [ ] Breadcrumb schema considered
