# Information Gain Checklist

## Core Question

```text
What does this page add that similar pages do not explain clearly?
```

## Experience

- [ ] The content includes real-world context
- [ ] The article mentions common situations from actual SEO work
- [ ] The page includes practical observations
- [ ] The content explains what usually goes wrong

## Specificity

- [ ] Examples are specific
- [ ] Advice is not generic
- [ ] The page includes clear conditions: when to use, when to avoid
- [ ] The page includes numbers, steps, categories, or decision criteria where useful

## Structure

- [ ] A named process or framework is used
- [ ] The page includes tables or checklists
- [ ] The structure helps users make decisions
- [ ] Each H2 has a clear job

## Verification

- [ ] The page explains how to verify important claims
- [ ] Tools are mentioned when useful
- [ ] Sources are added when needed
- [ ] Outdated claims are reviewed
