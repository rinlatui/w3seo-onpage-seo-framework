# Cannibalization and SERP Fit Checklist

## Primary Page

- [ ] Primary page is defined
- [ ] Supporting pages are defined
- [ ] Keyword target is unique enough
- [ ] Service page and blog page do not compete directly

## SERP Fit

- [ ] Page type matches the keyword intent
- [ ] Title matches the expected SERP format
- [ ] H1 is clear and unique
- [ ] CTA matches intent
- [ ] Content depth matches competitors but adds original value

## Internal Link Direction

- [ ] Supporting pages link to primary page
- [ ] Primary page links to useful supporting pages
- [ ] Anchor text is descriptive
- [ ] Redirected internal links are removed
