# WordPress Publishing Checklist

## Before Publishing

- [ ] Correct post category selected
- [ ] Tags used only when useful
- [ ] SEO title checked
- [ ] Meta description checked
- [ ] Featured image added
- [ ] Image alt text added
- [ ] Slug checked
- [ ] Table of contents reviewed

## Content

- [ ] H1 is not duplicated inside the editor
- [ ] H2/H3 hierarchy is clean
- [ ] FAQ section added if useful
- [ ] CTA added
- [ ] Internal links added
- [ ] External references checked

## Technical

- [ ] Canonical is correct
- [ ] Noindex is not accidentally enabled
- [ ] Page is included in sitemap if needed
- [ ] Schema does not conflict
- [ ] Mobile layout checked
