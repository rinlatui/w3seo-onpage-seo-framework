# Product or Category Page Template

## Main Keyword

```text
[Product/category keyword]
```

## H1

```text
[Product/category name]
```

## Recommended Structure

### Intro

- Define product/category
- Explain who needs it
- Add main keyword naturally

### Specification Section

- Size
- Material
- Standard
- Use case
- Buying notes

### Product List or Category Items

Show products clearly.

### Buying Guide

Explain how to choose correctly.

### FAQ

Answer common buying questions.

### CTA

Guide users to request quote, contact, or compare options.
