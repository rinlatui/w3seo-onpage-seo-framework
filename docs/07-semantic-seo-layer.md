# 07. Semantic SEO Layer

Semantic SEO focuses on meaning, entities, relationships, and topic coverage instead of keyword repetition.

## Core Elements

- Entity coverage
- Topical map
- Content hub
- Related questions
- Information gain
- Definition-led structure
- Fact density
- Contextual internal links

## Entity Coverage

For each topic, list related entities that should appear naturally.

Example topic:

```text
Onpage SEO
```

Related entities:

```text
Search Intent
Heading Structure
Internal Link
Meta Title
Meta Description
Schema Markup
Image SEO
Canonical Tag
WordPress
Google Search
EEAT
```

## Topical Map

Example:

```text
Onpage SEO Hub
├── Keyword Mapping
├── Heading Structure
├── Internal Linking
├── Image SEO
├── Schema Markup
└── WordPress Publishing Checklist
```

## Definition-Led Structure

Start important sections with a clear definition or short answer.

Example:

```text
Onpage SEO is the process of optimizing page-level elements such as title, headings, content, internal links, images, and schema so search engines and users can understand the page better.
```
