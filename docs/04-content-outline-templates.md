# 04. Content Outline Templates

A content outline turns keyword research and intent analysis into a writing structure.

## Outline Principles

- The first H2 should answer the main question.
- Each H2 should solve a distinct subtopic.
- H3 should support H2, not create unrelated ideas.
- FAQ should answer real user questions.
- CTA should match the user's intent.

## Blog Article Outline

```text
H1: Main topic

Introduction:
- User problem
- Experience context
- Promise of the article

H2: Short answer / definition
H2: Why this topic matters
H2: Main components
  H3: Component 1
  H3: Component 2
  H3: Component 3
H2: Step-by-step implementation
H2: Common mistakes
H2: Checklist
H2: FAQ
H2: Conclusion
```

## Service Page Outline

```text
H1: Service keyword + value proposition

Hero:
- Service promise
- Trust signal
- CTA

H2: What the service is
H2: Who needs this service
H2: Scope of work
H2: Process
H2: Proof / case study
H2: Pricing or package guidance
H2: FAQ
H2: Final CTA
```

## Author Page Outline

```text
H1: Author name

H2: Professional summary
H2: Expertise
H2: Experience
H2: Reviewed or published content
H2: Editorial role
H2: Profiles and sameAs links
H2: Contact
```
