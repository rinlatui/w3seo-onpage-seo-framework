# 05. Onpage SEO Checklist Explained

Onpage SEO is the process of improving page-level elements so users and search engines can understand the content clearly.

## Title

A good title should include the main keyword and a clear value proposition.

## Meta Description

A good meta description should explain what the page helps users do.

## URL Slug

The slug should be short, readable, and close to the main keyword.

## Heading Structure

Use one H1. Use H2 for main sections and H3 for supporting points.

## Introduction

The introduction should quickly confirm that the user is in the right place.

## Body Content

Each section should answer one clear need.

## Image SEO

Images should have descriptive file names, compressed size, and meaningful alt text.

## Internal Links

Internal links should connect related pages and support topical authority.

## External References

External references should support trust when facts, data, standards, or technical claims are mentioned.

## CTA

The CTA should match the user intent. Informational pages can use soft CTA. Service pages can use stronger CTA.
