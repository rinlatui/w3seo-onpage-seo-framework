# 15. Examples

This section provides example structures for common SEO page types.

## Blog Post Example

Topic:

```text
Onpage SEO là gì?
```

Structure:

```text
H1: Onpage SEO là gì? Hướng dẫn tối ưu nội dung và cấu trúc trang

H2: Onpage SEO là gì?
H2: Vì sao Onpage SEO quan trọng?
H2: Các yếu tố Onpage SEO quan trọng
  H3: Title và meta description
  H3: Heading structure
  H3: Internal link
  H3: Image SEO
  H3: Schema markup
H2: Checklist tối ưu Onpage SEO
H2: FAQ
H2: Kết luận
```

## Service Page Example

Topic:

```text
Dịch vụ SEO tổng thể
```

Structure:

```text
H1: Dịch vụ SEO tổng thể giúp website tăng trưởng bền vững

H2: Dịch vụ SEO tổng thể là gì?
H2: Khi nào doanh nghiệp cần SEO tổng thể?
H2: Quy trình triển khai SEO tổng thể
H2: Bảng hạng mục công việc
H2: FAQ
H2: Liên hệ tư vấn
```

## Author Page Example

Topic:

```text
Đoàn Trình Dục
```

Structure:

```text
H1: Đoàn Trình Dục

H2: Giới thiệu
```
