# 18. Conversion and CTA Layer

SEO content should not only bring traffic. It should guide users to the next useful action.

## CTA Types

| CTA Type | Best For |
|---|---|
| Soft CTA | Informational articles |
| Service CTA | Commercial/service pages |
| Consultation CTA | B2B and agency pages |
| Internal CTA | Content hubs |
| Lead Magnet | Checklists/templates |
| Contact Block | Local/service pages |
| Trust CTA | Author pages and EEAT pages |

## CTA by Intent

| Search Intent | CTA Style |
|---|---|
| Informational | Read related guide, download checklist |
| Commercial | Compare service, request consultation |
| Transactional | Contact, request quote, book audit |
| Local | Call, Zalo, map, contact form |
| Author/entity | View profile, see projects, connect |

## Trust Before CTA

A strong CTA works better after trust signals:

- Experience note
- Case study
- Author/reviewer proof
- Process explanation
- Checklist
- FAQ

## CTA Checklist

- [ ] CTA matches search intent
- [ ] CTA is visible but not spammy
- [ ] CTA text is specific
- [ ] CTA appears after value is delivered
- [ ] Service pages have direct contact CTA
- [ ] Informational pages have soft CTA
- [ ] Author pages have profile/contact CTA
