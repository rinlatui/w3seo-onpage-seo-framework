# 19. Content Quality Scoring Layer

The Content Quality Scoring Layer turns the framework into a measurable review system.

## Scoring Model

Score each item from 0 to 10.

| Category | Score |
|---|---|
| Search Intent Match | 0–10 |
| Keyword Mapping | 0–10 |
| Heading Structure | 0–10 |
| EEAT | 0–10 |
| Semantic SEO | 0–10 |
| Information Gain | 0–10 |
| Fact Density | 0–10 |
| Internal Linking | 0–10 |
| Schema Readiness | 0–10 |
| GEO / AI Extraction | 0–10 |
| UX & Readability | 0–10 |
| Conversion & CTA | 0–10 |
| Refresh Potential | 0–10 |
| Cannibalization Risk | 0–10 |

## Score Interpretation

| Total Score | Status |
|---|---|
| 0–50 | Needs major rewrite |
| 51–70 | Needs improvement before publishing |
| 71–85 | Publishable with minor improvements |
| 86–100 | Strong content |
| 100+ | Excellent content system |

## Example Review

```text
Search Intent Match: 8
Keyword Mapping: 7
Information Gain: 6
Internal Linking: 5
GEO Extraction: 7
CTA: 4

Action: improve CTA, add internal links, add original examples.
```
