# 11. Internal Linking Model

Internal links help users and search engines understand relationships between pages.

## Internal Link Principles

- Link from related content.
- Use descriptive anchor text.
- Avoid empty anchors.
- Avoid linking to redirected URLs.
- Link to important money pages from supporting articles.
- Link from hubs to clusters and from clusters back to hubs.

## Link Models

### Hub to Cluster

```text
Onpage SEO Guide → Heading Structure Guide
Onpage SEO Guide → Internal Linking Guide
Onpage SEO Guide → Image SEO Guide
```

### Cluster to Hub

```text
Heading Structure Guide → Onpage SEO Guide
Image SEO Guide → Onpage SEO Guide
```

### Article to Service Page

```text
Blog article → SEO audit service page
Blog article → WordPress optimization service page
```

### Author Entity Links

```text
Author page → GitHub profile
Author page → LinkedIn profile
Author page → Other author profiles
GitHub README → Main author profile
```

## Anchor Text Examples

Bad:

```text
click here
read more
this article
```

Better:

```text
technical SEO audit checklist
WordPress onpage SEO guide
EEAT author profile structure
```
