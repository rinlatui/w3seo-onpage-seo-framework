# 06. EEAT Layer

EEAT stands for Experience, Expertise, Authoritativeness, and Trustworthiness.

## Experience

Show real-world context.

Examples:

- Practical observations
- Before/after cases
- Mistakes found during audits
- Lessons from WordPress projects

## Expertise

Show technical understanding.

Examples:

- Clear frameworks
- Step-by-step processes
- Correct SEO concepts
- Structured recommendations

## Authoritativeness

Connect content to an author, reviewer, organization, and related works.

Examples:

- Author profile
- Reviewer box
- GitHub profile
- LinkedIn profile
- Related case studies
- Published works

## Trustworthiness

Help users verify the content.

Examples:

- Update date
- Source references
- Transparent limitations
- Checklist for self-verification
- Clear contact or author identity

## EEAT Placement

| Page Area | EEAT Signal |
|---|---|
| Introduction | Experience |
| Main content | Expertise |
| Author box | Authoritativeness |
| Reviewer box | Trust |
| Conclusion | Verification guidance |
