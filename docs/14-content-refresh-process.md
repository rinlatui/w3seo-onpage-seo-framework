# 14. Content Refresh Process

SEO content needs updates when search intent changes, rankings drop, or new information becomes important.

## When to Refresh Content

- Traffic drops
- Ranking drops
- Search intent changes
- Competitors add better sections
- Content becomes outdated
- Internal links are missing
- FAQ is thin
- Schema is missing or outdated

## Refresh Workflow

1. Check current traffic and keywords
2. Identify ranking pages and lost queries
3. Re-check search intent
4. Update outdated sections
5. Add missing subtopics
6. Add new FAQ
7. Improve headings
8. Add internal links
9. Update images if needed
10. Update sources or verification notes
11. Update modified date if appropriate
12. Re-submit URL if needed
