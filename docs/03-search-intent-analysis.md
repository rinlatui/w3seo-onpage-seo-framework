# 03. Search Intent Analysis

Search intent is the reason behind a user's query. Before writing, identify what the user really wants.

## Intent Types

| Intent Type | Meaning | Recommended Format |
|---|---|---|
| Informational | User wants to understand a topic | Guide, definition, explanation |
| Commercial | User is comparing options | Comparison, checklist, buying guide |
| Transactional | User wants to take action | Service page, landing page, CTA |
| Navigational | User wants a specific brand/page | Brand page, contact page, author page |
| Comparison | User compares A and B | Comparison table, pros and cons |
| Local | User wants local service/result | Local landing page, map, phone, local proof |
| Problem-solving | User has a specific issue | Step-by-step fix, troubleshooting guide |

## W3SEO Intent Questions

1. What does the user want to know?
2. What does the user need to decide?
3. What problem are they trying to solve?
4. What proof would make them trust the content?
5. What next action should they take?

## SERP Fit

A page should match the type of content Google already rewards, but should also add more useful value.

Example:

```text
Keyword: SEO onpage là gì
Intent: Informational
Best page type: guide article
CTA style: soft CTA
```

```text
Keyword: dịch vụ SEO onpage
Intent: Transactional / Commercial
Best page type: service page
CTA style: direct CTA
```
