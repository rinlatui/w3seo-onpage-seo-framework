# 01. W3SEO Process

The W3SEO process is a practical workflow for creating SEO content with clear structure, keyword strategy, useful information, and trust signals.

## The 5 Core Steps

1. Find the main keyword and LSI keywords
2. Build a detailed SEO outline
3. Write content based on the outline
4. Insert SEO keywords naturally
5. Improve EEAT with experience, expertise, authoritativeness, and trustworthiness

## Step 1: Keyword Research

Output:

- Main keyword
- LSI keywords
- Related entities
- User questions
- Search intent
- Content angle

## Step 2: Outline Building

A good outline defines the page before writing.

Required outline elements:

- H1
- Introduction angle
- H2 sections
- H3 supporting points
- FAQ section
- Internal link opportunities
- CTA placement
- Schema suggestion

## Step 3: Content Writing

Each section should solve a specific user need.

Recommended section pattern:

```text
Short answer → Explanation → Example → Checklist → Internal link
```

## Step 4: Keyword Placement

Keywords should improve clarity, not damage readability.

Recommended areas:

- SEO title
- Meta description
- URL slug
- H1
- First 100 words
- H2/H3 where natural
- Image alt text
- FAQ
- Conclusion

## Step 5: EEAT Improvement

Each content piece should show:

- Experience: practical examples and context
- Expertise: clear technical explanation
- Authoritativeness: author/reviewer/entity connection
- Trustworthiness: source, update date, verification guidance
