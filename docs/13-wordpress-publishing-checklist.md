# 13. WordPress Publishing Checklist

Publishing in WordPress should include content, SEO, UX, and technical checks.

## Before Publishing

- Select correct category
- Avoid unnecessary tags
- Check SEO title
- Check meta description
- Check slug
- Add featured image
- Add image alt text
- Check H2/H3 structure
- Add internal links
- Add CTA
- Check FAQ section

## Technical Checks

- Canonical is correct
- Noindex is not accidentally enabled
- Sitemap includes the URL if needed
- Schema does not conflict
- Mobile layout works
- Page speed is acceptable
- Links do not redirect unnecessarily

## After Publishing

- Open the live URL
- Test internal links
- Check images
- Inspect URL if needed
- Add internal links from older related pages
- Monitor impressions and clicks
