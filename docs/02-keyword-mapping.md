# 02. Keyword Mapping

Keyword mapping means assigning each keyword type to the correct page element.

## Keyword Types

| Type | Meaning | Example |
|---|---|---|
| Main keyword | Core ranking target | onpage SEO |
| LSI keyword | Supporting related term | heading structure, meta description |
| Entity keyword | Recognized concept/object | Google Search, WordPress, Schema Markup |
| Question keyword | User question | Onpage SEO gồm những gì? |
| Commercial keyword | Buying/service intent | dịch vụ SEO onpage |

## Keyword Placement Map

| Page Element | Recommended Keyword |
|---|---|
| SEO Title | Main keyword |
| URL Slug | Main keyword or short variation |
| H1 | Main keyword |
| Introduction | Main keyword + user problem |
| H2 | LSI keyword or subtopic |
| H3 | Supporting keyword |
| Body | Natural keyword variations |
| FAQ | Question keywords |
| CTA | Commercial keyword |
| Image Alt | Descriptive keyword where natural |
| Conclusion | Main keyword + trust signal |

## Example

Main keyword:

```text
dịch vụ SEO tổng thể
```

Mapping:

```text
Title: Dịch vụ SEO tổng thể giúp website tăng trưởng bền vững
H1: Dịch vụ SEO tổng thể
H2: Dịch vụ SEO tổng thể là gì?
H2: Quy trình triển khai SEO tổng thể
FAQ: Dịch vụ SEO tổng thể gồm những gì?
CTA: Liên hệ tư vấn dịch vụ SEO tổng thể
```

## Common Mistakes

- Using the main keyword too many times
- Ignoring question keywords
- Using unrelated LSI keywords
- Adding keywords into headings unnaturally
- Forgetting commercial keywords on service pages
