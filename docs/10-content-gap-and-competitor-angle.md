# 10. Content Gap and Competitor Angle

Content Gap Analysis helps identify what a page is missing compared to user needs and competing results.

In W3SEO, content gap is not only about copying competitor headings. It is about finding missing value.

## Content Gap Types

| Gap Type | Meaning |
|---|---|
| Intent gap | Page does not answer the real user intent |
| Topic gap | Important subtopics are missing |
| Entity gap | Important related entities are missing |
| Example gap | Page has no practical examples |
| Trust gap | Page lacks author, reviewer, sources, or verification |
| Conversion gap | Page has no clear next step |
| Freshness gap | Page uses outdated information |
| Local gap | Page ignores local context when local intent exists |

## Competitor Review Questions

1. What sections appear repeatedly in top results?
2. What questions do users need answered?
3. What examples are missing from most competitors?
4. What mistakes are competitors not explaining?
5. What original framework can this page add?
6. What proof or verification can make this page more trustworthy?
